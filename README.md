# JavaScript Learning

Welcome to the JavaScript Learning repository! This collection is designed as concise study material for mastering JavaScript fundamentals and advanced concepts.

## Contents
- **data-types/**: Explore JavaScript primitive types and their methods.
- **objects/**: Learn about object creation, constructors, garbage collection, and the `this` keyword. Also Understand the symbol type and its usage.
## How to Use
Browse the folders and files for code examples, notes, and tasks. Each section is organized to help you build a strong foundation and deepen your understanding of JavaScript.
